# 🌐 Personal Portfolio Website

This is my **second HTML & CSS project** — a personal portfolio website built to practice frontend development and improve my design skills.

It showcases who I am, what I’m learning, and the projects I’ve built so far as a beginner web developer.

---

## 📸 Preview

👉 https://zaydelm.github.io/portfolio/

---

## 🧠 About This Project

I built this portfolio to get more comfortable with structuring web pages using HTML and styling them with CSS.

The goal was to create something clean, simple, and responsive while learning real-world layout techniques like Flexbox and Grid.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Syne & DM Sans)

---

## 🎯 Features

- Clean dark-themed design
- Responsive layout (works on most screen sizes)
- Smooth navigation with anchor links
- Skills and project showcase sections
- Simple contact section
- Hover effects for better interactivity
